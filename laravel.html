Pengenalan dan Instalasi Laravel

Apa itu Laravel?

Laravel adalah framework aplikasi web PHP yang open-source. Mengikuti pola desain Model-View-Controller (MVC). Bertujuan untuk membuat pengembangan web lebih mudah, cepat, dan menyenangkan. Menyediakan berbagai fitur siap pakai untuk pengembangan aplikasi web modern.

Kebutuhan Instalasi

  • Siapkan Composer
  • Node.js
  • VSCode/Visual Studio Code

Struktur Direktori

  • app/: Berisi kode aplikasi (model, controller, middleware).
  • routes/: Mendefinisikan rute aplikasi.
  • resources/views/: Berisi file tampilan (Blade templates).
  • database/: Berisi migration dan seeder database.
  • public/: Direktori publik untuk aset (CSS, JavaScript, gambar).
  • config/: Berisi file konfigurasi aplikasi.
Cara Instal
  1. Install Laravel menggunakan Composer: 
    composer create-project laravel/laravel nama_proyek
  2. Masuk ke direktori proyek: 
    cd nama_proyek
  3. Jalankan server pengembangan Laravel: 
    php artisan serve
  4. Buka browser dan akses: 
    http://127.0.0.1:8000
